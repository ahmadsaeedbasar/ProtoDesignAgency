# Contributing

## Prerequisites

- Node.js 20+
- npm 10+

## Setup

1. `npm ci`
2. `npm run dev`

## Pull Requests

- Keep PRs focused and small.
- Ensure `npm run build` passes before opening a PR.
- Update docs when behavior/configuration changes.

## Commit Style

Conventional Commits are recommended:

- `feat: ...`
- `fix: ...`
- `docs: ...`
- `chore: ...`
